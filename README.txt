Hierarchive: a hierarchical archive management system.

(It's not ready for use, yet.)

INSTALLATION

1. Rename or copy config.dist.php to config.php, and htaccess.dist to .htaccess
2. Edit both of these files to include the correct details
3. Run mysql_schema.sql against your database
